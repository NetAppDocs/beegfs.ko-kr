= BeeGFS CSI 드라이버 구성
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
BeeGFS CSI 드라이버를 사용하면 Kubernetes 클러스터가 BeeGFS 볼륨을 동적으로 프로비저닝하고 관리하고, 고성능 BeeGFS 스토리지를 컨테이너화된 워크로드와 통합할 수 있습니다.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["하나"]모든 전제 조건이 충족되었는지 확인하세요
[role="quick-margin-para"]
필수 구성 요소를 검토하여 실행 중인 Kubernetes 클러스터, BeeGFS 클러스터에 대한 연결, 클러스터 전체 리소스를 설치하는 데 필요한 권한이 있는지 확인하세요.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["둘"]BeeGFS CSI 드라이버 배포
[role="quick-margin-para"]
BeeGFS CSI 드라이버 저장소를 복제하고 제공된 Kubernetes 매니페스트나 오버레이를 사용하여 드라이버를 배포합니다.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["삼"]배포 확인
[role="quick-margin-para"]
관련 Kubernetes Pod를 검사하여 모든 BeeGFS CSI 드라이버 구성 요소가 설치되고 작동하는지 확인하세요.  노드 스케줄링 충돌이나 노드에서 스왑이 활성화되는 등의 문제를 해결합니다.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-4.png["네"]예제 애플리케이션 배포 및 검증
[role="quick-margin-para"]
Kubernetes 매니페스트 예제를 적용하고 워크로드가 예상대로 BeeGFS 지원 스토리지에 액세스할 수 있는지 확인합니다.
